# 100億円タイプ診断 質問集（20問）

全質問が100億円テーマに統一された拡張版。4軸（リスク・時間・判断・価値）に対応し、一部は「100億円の分配選択式」でゲーム性と診断精度を高めています。

---

## 回答形式

- **5段階**：1＝全くそう思わない、2＝そう思わない、3＝どちらとも、4＝そう思う、5＝強くそう思う
- **分配選択**：A/B/C/D から1つ選択

---

## リスク軸（S ↔ R）Q1〜Q5

安定志向（S）と挑戦志向（R）を測定。高スコアほど R 寄り。

### Q1
**100億円のうち50億円を、成功すれば10倍になるが失敗すればゼロになる事業に投資する。**

- 形式：5段階（1〜5点）
- 加点方向：R
- 備考：そのまま加算

### Q2（逆転項目）
**100億円は安定資産に分散し、年間数％で確実に増やす方針にする。**

- 形式：5段階（1〜5点 → **6−回答値** で加算）
- 加点方向：R（逆転のため、低同意＝高 R）
- 備考：同意が高いほど S 寄りなので、スコアは反転させる

### Q3（分配形式）
**100億円を安定運用と成長投資にどう配分しますか？**

| 選択肢 | 配分 | 得点 |

| A | 80億 安定運用 / 20億 成長投資 | 0 |
| B | 60億 安定 / 40億 成長 | 1 |
| C | 40億 安定 / 60億 成長 | 3 |
| D | 20億 安定 / 80億 成長 | 5 |

- 加点方向：R
- 備考：成長寄りほど高得点

### Q4（逆転項目）
**100億円で元本保証に近い方法以外は基本選ばない。**

## 配点サマリー

| 軸 | 質問番号 | 通常 | 逆転 | 分配 | 最大 |
|----|----------|------|------|------|------|
| リスク | Q1〜Q5 | Q1, Q5 | Q2, Q4 | Q3 | 25点 |
| 時間 | Q6〜Q10 | Q7, Q9 | Q6, Q10 | Q8 | 25点 |
| 判断 | Q11〜Q15 | Q12, Q14 | Q11, Q15 | Q13 | 25点 |
| 価値 | Q16〜Q20 | Q17, Q19 | Q16, Q20 | Q18 | 25点 |

- **通常**：回答値 1〜5 をそのまま加算
- **逆転**：6−回答値 を加算（同意が高いほど右極のスコアは低く）
- **分配**：A=0, B=1, C=3, D=5 を加算

---

## 実装時の注意

- 5段階質問の未回答は 0 扱い（または必須にして未入力のまま送信不可にする）
- 分配質問は A/B/C/D のいずれか必須
- 逆転項目はサーバー側またはフロントで `6 - value` を計算してから加算

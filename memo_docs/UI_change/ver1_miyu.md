# 診断アプリUI改善要望

## 1. 設問カードのデザイン統一
現在は **Q1～Q3のみカードの端に色が付いている状態**になっている。  
この **カード端の色を削除し、Q1～Q20すべての設問カードの見た目を統一**したい。

---

## 2. 回答中の質問の強調表示
**現在回答している質問カードのみ明るく表示して強調**する。  

それ以外の質問カードは **少し暗めに表示**し、  
ユーザーが **どの質問に答えればよいか分かりやすいUI**にしたい。

---

## 3. 自動スクロール機能
ユーザーが回答を選択したら、  
**自動で次の質問へスクロールして進むようにする。**

これにより、ユーザーが **順番に回答しやすいUI**にする。

---

## 4. デザインの雰囲気
- **エンタメ系Webアプリらしいポップな色合い**にする  
- **シンプルなデザインは維持する**  
- **フォントは現状のままで問題なし**

## FPTI との対応関係

本診断で参照している理論・モデルと文献の対応は以下の通りです。

| FPTI 要素 | 参照文献・モデル |
|-----------|------------------|
| リスク態度（S/R） | BIT、Prospect Theory、リスク許容度研究 |
| 時間志向（L/N） | MAS（Retention-Time）、Big Five |
| 意思決定スタイル（A/I） | BIT、認知・感情バイアス研究 |
| 価値の向き（P/C） | KMSI、Big Five（外向性・協調性）、MAS（Power-Prestige） |
| 4タイプ（BIT） | Pompian (2012) |
| マネースクリプト4種 | Klontz et al. (2011) |
| 金融態度5次元 | Yamauchi & Templer (1982) |
| 性格×金融行動 | Exley et al. (2021), De Bortoli et al. (2019) |
